# Merkle Distributor Airdrop

## NFT merkle airdrop
本样例介绍了 5 种 NFT 空投方式

直接对特定账户进行空投

线下签名,线上验证通过后空投

线下 EIP-712 方式签名, 线上验证后空投

线下 EIP-712 方式签名, 线上进行 EIP-712 验证,Signature Check

线下生成 Merkle 证明, 线上 Merkle 验证

## ERC20 merkle airdrop
参考 1inch，dydx，uniswap 都实现 merkle 空投。
